Detailed Description

template<typename T>
class wmtk::attribute::TupleAccessor< T >

A CachingAccessor that uses tuples for accessing attributes instead of indices. As global simplex ids should not be publicly available, this accessor uses the Mesh.id() function to map from a tuple to the global simplex id.
