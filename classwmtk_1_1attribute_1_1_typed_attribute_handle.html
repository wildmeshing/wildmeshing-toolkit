Detailed Description

template<typename T>
class wmtk::attribute::TypedAttributeHandle< T >

Handle that represents attributes for some mesh.

In conjunction with the mesh that constructed it, specifies a particular attribute. NOTE: with multimesh around, you ideally want to store a MeshAttribteHandle, which also hides the templating used here.
