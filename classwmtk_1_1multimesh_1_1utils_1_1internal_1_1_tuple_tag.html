Detailed Description

TupleTag is a util helper class for tagging edges in a triangle mesh.

These tags can then be used to extract edge meshes as child mesh for multimesh.

The tagging algorithm is based on the union-find algorithm. It first tags all vertices by grouping them into disjoint sets. Then it creates edge tags based on the vertex tags. Vertex tag and edge tags are stored as mesh attributes of the parent triangle mesh, and can be accessed through "vertex_tag_handle" and "edge_tag_handle" respectively.
