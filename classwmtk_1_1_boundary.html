void wmtk::Boundary::construct_boundaries ( const Eigen::MatrixXd &  V,
const Eigen::MatrixXi &  F,
const Eigen::MatrixXi &  E0,
const Eigen::MatrixXi &  E1 
)

Computes a list of parameterized curves from the boundary loops of a mesh. Optional seam edges can be provided, causing loops to be split at T-junctions between seams. Curves across the same seam will have a "parent curve" defined between the two of them.

Note
IMPORTANT Vertices on the left column of E0 must match vertices on the left column of E1. I.e. the orientation matters for E0 and E1.
Parameters
[in]V#V x 2 matrix of mesh vertices.
[in]F#F x 3 matrix of mesh faces.
[in]E0#E x 2 matrix of seam edges (left side).
[in]E1#E x 2 matrix of seam edges (right side).
